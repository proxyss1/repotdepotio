Process
How it works

Four steps from App ID to a ready-to-use SteamTools manifest file.

01 / LOOKUP

App resolution

The App ID is resolved against the Steam store API to retrieve the game name and metadata.

02 / DEPOTS

Depot discovery

All Windows-compatible depots are fetched from the SteamCMD API. Shared and non-Windows depots are filtered automatically.

03 / KEYS

Key matching

Each depot ID is matched against the HomeDepot decryption key database. Only depots with valid keys are included.

04 / OUTPUT

Manifest download

A fully formed .lua file is generated and downloaded — ready to drag into SteamTools.
